# TaskManager-api

## Начало

1. Скачать репозиторий:

    ```bash
    git clone https://github.com/StuffCod3/TaskManager-api.git
    ```

2. Переход в директорию проекта:

    ```bash
    cd TaskManager
    ```

3. Запуск докер контейнера:

    ```bash
    docker-compose up
    ```

4. Доступ к приложению [http://localhost:8080](http://localhost:8080).
